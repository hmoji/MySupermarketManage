# MySupermarketManage
以ssm框架搭建的超市管理系统

一个用来学习的小项目。

*这配置文件能要我半条命~~~*

- 项目管理工具是maven
- 服务器是tomcat
- 数据库是mysql
- [x] spring和mybatis的连接
- [x] spring整合mybatis的配置
- [x] springMVC的配置
- [x] ssm基本配置
- [x] 通过测试
- [x] 编写基本业务，实现参数的前后端传递
- [ ] 权限管理
- [ ] 更多业务逻辑······
